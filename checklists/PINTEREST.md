# Pinterest Business Profile checklist

## 🔹 Pinterest Business Profile

### 📂 Account Setup
- [ ] Username is short, branded, and consistent with other platforms  
- [ ] Profile picture (logo) uploaded in high resolution  
- [ ] Bio includes brand tagline, keywords, and call-to-action (CTA)  
- [ ] Website claimed and verified (important for analytics & authority)  
- [ ] Business account enabled (not personal)  
- [ ] Category selected (e.g., Retail, Lifestyle, Tech, Education)  
- [ ] Connected to other social accounts (Instagram, YouTube, etc.)  

### 🎯 Content & Engagement
- [ ] Regular pinning schedule (minimum 5–10 pins/week)  
- [ ] Boards created and organized by themes/products/services  
- [ ] Pins optimized with keyword-rich titles and descriptions  
- [ ] High-quality vertical images (2:3 ratio) used consistently  
- [ ] Rich Pins enabled (product, article, recipe)  
- [ ] Engagement tracked (saves, clicks, comments)  
- [ ] Community interaction (responding to comments/messages)  

### 📊 Ads & Promotions
- [ ] Pinterest Ads Manager connected  
- [ ] Audience targeting set (interests, demographics, keywords)  
- [ ] Promoted Pins tested (static, carousel, video)  
- [ ] Conversion tracking installed on website  
- [ ] A/B testing creatives and formats  
- [ ] ROAS (Return on Ad Spend) monitored  

### 🔐 Security & Authority
- [ ] Two-factor authentication enabled  
- [ ] Verified merchant badge applied (if eligible)  
- [ ] Admin roles managed securely  
- [ ] Copyright compliance checked (no reused or unlicensed images)  

### 📈 Analytics & Reporting
- [ ] Pinterest Analytics reviewed weekly (impressions, clicks, saves)  
- [ ] Top-performing pins identified and replicated  
- [ ] Audience demographics analyzed (age, gender, location, interests)  
- [ ] Traffic sources tracked (search, boards, external shares)  
- [ ] Conversion tracking set up (website clicks, leads, sales)
