# TikTok Business Account checklist

## 🔹 TikTok Business Account

### 📂 Account Setup
- [ ] Username is short, branded, and consistent with other platforms  
- [ ] Profile picture (logo) uploaded in high resolution  
- [ ] Bio includes brand tagline, keywords, and call-to-action (CTA)  
- [ ] Website link added (with UTM tracking if possible)  
- [ ] Business account enabled (not personal)  
- [ ] Category selected (e.g., Retail, Tech, Education)  
- [ ] Connected to Ads Manager for campaigns  

### 🎯 Content & Engagement
- [ ] Regular posting schedule (minimum 3–5 videos/week)  
- [ ] Videos optimized for vertical format (9:16)  
- [ ] Use trending sounds and hashtags for reach  
- [ ] Captions include CTAs and relevant keywords  
- [ ] Engagement tracked (likes, comments, shares, saves)  
- [ ] Duets and stitches used for community interaction  
- [ ] Replies to comments and DMs within 24 hours  

### 📊 Ads & Promotions
- [ ] TikTok Ads Manager connected  
- [ ] Pixel installed on website for conversion tracking  
- [ ] Custom audiences created (website visitors, followers)  
- [ ] Lookalike audiences tested for scaling campaigns  
- [ ] Spark Ads (boosting organic posts) tested  
- [ ] A/B testing creatives and formats  
- [ ] ROAS (Return on Ad Spend) monitored  

### 🔐 Security & Authority
- [ ] Two-factor authentication enabled  
- [ ] Verified badge applied (if eligible)  
- [ ] Admin roles managed securely  
- [ ] Linked Instagram/YouTube accounts for cross-promotion  

### 📈 Analytics & Reporting
- [ ] TikTok Analytics reviewed weekly (views, engagement, followers)  
- [ ] Top-performing videos identified and replicated  
- [ ] Audience demographics analyzed (age, gender, location)  
- [ ] Traffic sources tracked (For You page, followers, hashtags)  
- [ ] Conversion tracking set up (website clicks, leads, sales)
