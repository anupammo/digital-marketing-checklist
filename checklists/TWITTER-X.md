# Twitter/X

## 🔹 Twitter/X Account

### 📂 Account Setup
- [ ] Handle (username) is short, branded, and consistent across platforms  
- [ ] Profile picture (logo) uploaded in high resolution  
- [ ] Header image optimized for desktop & mobile  
- [ ] Bio includes brand tagline, keywords, and call-to-action (CTA)  
- [ ] Website link added (with UTM tracking if possible)  
- [ ] Location and category filled (if applicable)  
- [ ] Verified badge applied (if eligible)  

### 🎯 Content & Engagement
- [ ] Regular posting schedule (minimum 1–2 tweets/day)  
- [ ] Mix of content types (text, images, GIFs, videos, polls)  
- [ ] Use of hashtags for discoverability  
- [ ] Engagement tracked (likes, comments, retweets, shares)  
- [ ] Replies to mentions and DMs within 24 hours  
- [ ] Threads used for storytelling and thought leadership  
- [ ] Participation in trending topics (when relevant)  

### 📊 Ads & Promotions
- [ ] Twitter Ads Manager connected  
- [ ] Audience targeting set (interests, demographics, keywords)  
- [ ] Promoted tweets tested (static, carousel, video)  
- [ ] Conversion tracking installed on website  
- [ ] A/B testing creatives and formats  
- [ ] ROAS (Return on Ad Spend) monitored  

### 🔐 Security & Authority
- [ ] Two-factor authentication enabled  
- [ ] Admin roles managed securely  
- [ ] Verified badge applied (if eligible)  
- [ ] Linked Instagram/LinkedIn accounts for cross-promotion  

### 📈 Analytics & Reporting
- [ ] Analytics reviewed weekly (impressions, engagement, followers)  
- [ ] Top-performing tweets identified and replicated  
- [ ] Audience demographics analyzed (age, location, interests)  
- [ ] Traffic sources tracked (search, hashtags, external shares)  
- [ ] Conversion tracking set up (website clicks, leads, sales)
