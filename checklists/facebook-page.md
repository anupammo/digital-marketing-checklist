# ✅ Social Media Account Checklist

## 🔹 Facebook Page & Ads Manager

### 📂 Account Setup
- [ ] Page name matches brand/business name  
- [ ] Profile picture (logo) is clear and high-resolution  
- [ ] Cover photo optimized for desktop & mobile  
- [ ] Username (URL slug) is short, branded, and consistent  
- [ ] About section filled with accurate business info  
- [ ] Contact details (phone, email, website) are correct  
- [ ] Location & hours updated (if applicable)  

### 🎯 Content & Engagement
- [ ] Regular posting schedule (minimum 2–3 posts/week)  
- [ ] Posts include mix of text, images, video, and links  
- [ ] Use of Facebook Stories & Reels for reach  
- [ ] Engagement rate (likes, comments, shares) tracked  
- [ ] Community management (replying to comments/messages)  

### 📊 Ads Manager
- [ ] Business Manager account verified  
- [ ] Pixel installed on website for tracking conversions  
- [ ] Custom audiences created (website visitors, email list)  
- [ ] Lookalike audiences tested for scaling  
- [ ] Ad creatives optimized for mobile-first  
- [ ] A/B testing campaigns running  
- [ ] ROAS (Return on Ad Spend) monitored  

### 🔐 Security & Authority
- [ ] Page verified (blue/gray badge if eligible)  
- [ ] Two-factor authentication enabled for admins  
- [ ] Admin roles assigned correctly (no unnecessary access)  
- [ ] Linked Instagram account for cross-posting  

### 📈 Analytics & Reporting
- [ ] Insights reviewed weekly (reach, engagement, followers)  
- [ ] Top-performing posts identified and replicated  
- [ ] Audience demographics analyzed (age, location, interests)  
- [ ] Conversion tracking set up (website clicks, leads, sales)
