## 🔹 WhatsApp Business Account

### 📂 Account Setup
- [ ] Business account enabled (not personal)  
- [ ] Profile picture (logo) uploaded in high resolution  
- [ ] Business name matches brand/business name  
- [ ] Business description includes tagline, keywords, and CTA  
- [ ] Contact details verified (phone number, email, website)  
- [ ] Business hours set correctly  
- [ ] Catalog created (if selling products/services)  

### 🎯 Content & Engagement
- [ ] Automated greeting message enabled  
- [ ] Away message configured for off-hours  
- [ ] Quick replies set up for FAQs  
- [ ] Broadcast lists used for announcements  
- [ ] Status updates posted regularly (offers, updates, tips)  
- [ ] Engagement tracked (message open rate, replies)  

### 📊 Ads & Promotions
- [ ] Connected to Facebook/Instagram Ads Manager  
- [ ] Click-to-WhatsApp ads tested  
- [ ] Conversion tracking installed on website  
- [ ] A/B testing creatives and CTAs  
- [ ] ROAS monitored for campaigns  

### 🔐 Security & Authority
- [ ] Two-factor authentication enabled  
- [ ] Verified badge applied (if eligible)  
- [ ] Admin roles managed securely  
- [ ] Linked Facebook/Instagram accounts for cross-promotion  

### 📈 Analytics & Reporting
- [ ] Message analytics reviewed weekly (delivery, read, response rates)  
- [ ] Top-performing broadcasts identified and replicated  
- [ ] Customer demographics analyzed (location, engagement patterns)  
- [ ] Conversion tracking set up (website clicks, leads, sales)
