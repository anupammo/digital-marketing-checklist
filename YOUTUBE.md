# YouTube Channel checklist

## 🔹 YouTube Channel

### 📂 Account Setup
- [ ] Channel name matches brand/business name  
- [ ] Profile picture (logo) uploaded in high resolution  
- [ ] Channel banner optimized for desktop, mobile, and TV  
- [ ] Channel description includes keywords, brand story, and links  
- [ ] Custom channel URL set (short, branded)  
- [ ] Links added to website and other social media accounts  
- [ ] Playlists created to organize content by topic/product/service  

### 🎯 Content & Engagement
- [ ] Regular upload schedule (minimum 1–2 videos/week)  
- [ ] Videos optimized with titles, descriptions, and tags  
- [ ] Thumbnails are branded, high-quality, and click‑worthy  
- [ ] Use of Shorts for short-form content reach  
- [ ] End screens and cards added for cross-promotion  
- [ ] Captions/subtitles included for accessibility and SEO  
- [ ] Engagement tracked (likes, comments, shares, watch time)  
- [ ] Community tab posts used for updates and polls  

### 📊 Ads & Monetization
- [ ] YouTube Ads campaigns tested (skippable, non-skippable, bumper ads)  
- [ ] Retargeting audiences set up via Google Ads  
- [ ] Monetization enabled (if eligible)  
- [ ] Memberships, Super Chat, or affiliate links integrated  
- [ ] A/B testing thumbnails and titles for CTR improvement  

### 🔐 Security & Authority
- [ ] Two-factor authentication enabled for channel admins  
- [ ] Channel verified with Google  
- [ ] Brand account used for multiple admin access  
- [ ] Copyright compliance checked (no reused or unlicensed content)  

### 📈 Analytics & Reporting
- [ ] YouTube Analytics reviewed weekly (views, watch time, subscribers)  
- [ ] Audience demographics analyzed (age, location, device)  
- [ ] Top-performing videos identified and replicated  
- [ ] Traffic sources tracked (search, suggested, external)  
- [ ] Conversion tracking set up (website clicks, leads, sales)
