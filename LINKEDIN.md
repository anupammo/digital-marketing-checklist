# LinkedIn Company Page checklist

## 🔹 LinkedIn Company Page

### 📂 Account Setup
- [ ] Company name matches official brand/business name  
- [ ] Logo uploaded (high-resolution, square format)  
- [ ] Banner image optimized for desktop & mobile  
- [ ] About section filled with mission, vision, and services  
- [ ] Website link added with UTM tracking (if possible)  
- [ ] Industry, company size, and specialties defined  
- [ ] Location(s) listed accurately  
- [ ] Custom LinkedIn URL set (short, branded)  

### 🎯 Content & Engagement
- [ ] Regular posting schedule (minimum 2–3 posts/week)  
- [ ] Mix of content types (articles, images, videos, polls)  
- [ ] Employee advocacy encouraged (staff sharing posts)  
- [ ] Engagement rate tracked (likes, comments, shares)  
- [ ] Use of LinkedIn Stories and Newsletters (if available)  
- [ ] Thought leadership posts from executives linked to page  

### 📊 Ads & Promotions
- [ ] LinkedIn Campaign Manager connected  
- [ ] Audience targeting set (job titles, industries, locations)  
- [ ] Sponsored content tested (posts, videos, carousel ads)  
- [ ] Lead Gen Forms enabled for campaigns  
- [ ] A/B testing creatives and formats  
- [ ] Conversion tracking installed on website  

### 🔐 Security & Authority
- [ ] Page admins assigned correctly (no unnecessary access)  
- [ ] Verified company status (if eligible)  
- [ ] Two-factor authentication enabled for admins  
- [ ] Showcase Pages created for sub-brands/products  

### 📈 Analytics & Reporting
- [ ] Page analytics reviewed weekly (visitors, followers, engagement)  
- [ ] Top-performing posts identified and replicated  
- [ ] Audience demographics analyzed (job function, seniority, location)  
- [ ] Follower growth tracked month-over-month  
- [ ] Conversion tracking set up (website clicks, leads, sales)
